| print()                                            | println                                                                   |
|----------------------------------------------------|---------------------------------------------------------------------------|
| print() function is a method of PrintStream Class. | println() function is a method of PrintStream Class.                      |
| print() function is only print given **String**    | println() function is print given **String** & end with new line (**\n**) |
